***********
 Reference
***********

The :py:mod:`aspen` Python package is divided into three major subpackages: the
:py:mod:`~aspen.simplates` file format, an HTTP
:py:mod:`~aspen.request_processor`, and a partial :py:mod:`~aspen.http`
implementation. In addition, there are supporting modules for
:py:mod:`~aspen.configuration`, :py:mod:`~aspen.exceptions`, and
:py:mod:`~aspen.utils`.
