The ^L character is an ASCII form feed, also called:

  • page break
  • \f
  • 12
  • 0xC

Aspen uses ^L to split simplates and the hooks.conf file into sections. If memory serves, I first saw it in Barry Warsaw's email package in the standard library. I think this means Barry uses Emacs, though I could be wrong.

Here is how to type ^L in various editors:

Vim Ctrl-L (in insert mode)
Emacs C-q C-l
UltraEdit If I remember right, there is a "Page Break" option on the "Insert" menu. You’d probably want to remap that in the long term.
TextMate, etc. ???

Please email me if you have any additions to that list or if you need help figuring it out for your editor.
